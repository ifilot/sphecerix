=========================================

:program:`Sphecerix` is a Python package for rotation of spherical harmonics using Wigner-D matrices

:program:`Sphecerix` has been developed at the Eindhoven University of Technology,
Netherlands. :program:`Sphecerix` and its development are hosted on `github
<https://github.com/ifilot/sphecerix>`_.  Bugs and feature
requests are ideally submitted via the `github issue tracker
<https://github.com/ifilot/sphecerix/issues>`_.
